# Public Example Sources

The images under [`examples/manifest-input/images`](./manifest-input/images) are public indoor equirectangular panoramas that can be redistributed with this repository.

Dataset summary:

- Scene: `San Giovanni Battista Martire (Montevecchia) - Interior`
- Source: Wikimedia Commons
- Author: Kaitu
